Проект задумывался как место, где можно собрать инструменты, полезных в процессе разработки. На данный момент имеется инструмент **[ttools/build](https://github.com/TuleshDev/ttools/tree/main/build)**, позволяющий построить проект с помощью копирования из разных источников. Его описание содержится в файле [ttools/build/README.md](https://github.com/TuleshDev/ttools/blob/main/build/README.ru.md).
